## 功能概览

### 核心功能
- ✅ **多搜索引擎支持**：Google、百度、Bing、DuckDuckGo、GitHub、Stack Overflow、YouTube、知乎
- ✅ **智能切换**：点击左侧图标展开引擎列表，内部滑动切换
- ✅ **实时搜索**：支持 Enter 键提交，自动编码搜索关键词
- ✅ **清除功能**：输入内容后显示清除按钮
- ✅ **悬停提示**：鼠标悬停显示搜索引擎名称
- ✅ **动画效果**：流畅的展开/收起动画

### UI 特点
- 🎨 **毛玻璃效果**：半透明背景 + 模糊效果
- 🎯 **图标可视化**：每个搜索引擎专属图标
- ✨ **悬停反馈**：按钮放大、阴影效果
- 📱 **响应式布局**：桌面、平板、手机完美适配

### 工具函数

```typescript
// 获取搜索引擎
export const getSearchEngine = (id: string): SearchEngine | undefined => {
  return searchEngines.find(engine => engine.id === id)
}

// 获取默认搜索引擎
export const getDefaultSearchEngine = (): SearchEngine => {
  return searchEngines[0]  // Google
}

// 执行搜索
export const performSearch = (query: string, engineId: string): void => {
  const engine = getSearchEngine(engineId)
  if (engine && query.trim()) {
    const searchUrl = engine.url + encodeURIComponent(query.trim())
    window.open(searchUrl, '_blank')
  }
}
```

### 2. 脚本部分

```vue
<script setup lang="ts">
import { ref, computed } from 'vue'
import { searchEngines, getSearchEngine, performSearch, type SearchEngine } from '@/utils/searchEngines'

// 状态
const searchQuery = ref('')
const selectedSearchEngine = ref('google')
const showDropdown = ref(false)
const hoveredEngine = ref<string | null>(null)

// 当前搜索引擎
const currentSearchEngine = computed(() => {
  return getSearchEngine(selectedSearchEngine.value) || searchEngines[0]
})

// 切换下拉菜单
const toggleDropdown = () => {
  showDropdown.value = !showDropdown.value
}

// 选择搜索引擎
const selectEngine = (engine: SearchEngine) => {
  selectedSearchEngine.value = engine.id
  showDropdown.value = false
}

// 处理搜索提交
const handleSearchSubmit = () => {
  if (searchQuery.value.trim()) {
    performSearch(searchQuery.value, selectedSearchEngine.value)
  }
}

// 清除搜索
const clearSearch = () => {
  searchQuery.value = ''
}

// 悬停处理
const handleEngineHover = (engine: SearchEngine) => {
  hoveredEngine.value = engine.name
}

const handleEngineLeave = () => {
  hoveredEngine.value = null
}

// 点击外部关闭下拉菜单
const handleClickOutside = (event: Event) => {
  const target = event.target as HTMLElement
  if (!target.closest('.search-engine-selector-left') && 
      !target.closest('.search-main-area')) {
    showDropdown.value = false
  }
}

// 添加全局点击事件监听
import { onMounted, onUnmounted } from 'vue'

onMounted(() => {
  document.addEventListener('click', handleClickOutside)
})

onUnmounted(() => {
  document.removeEventListener('click', handleClickOutside)
})
</script>
```

## React 实现

### 1. 组件代码

```tsx
import React, { useState, useEffect, useRef } from 'react';
import './SearchBox.css';

interface SearchEngine {
  id: string;
  name: string;
  icon: React.ReactNode;
  url: string;
  placeholder: string;
}

const searchEngines: SearchEngine[] = [
  {
    id: 'google',
    name: 'Google',
    icon: <GoogleIcon />,
    url: 'https://www.google.com/search?q=',
    placeholder: '在Google中搜索...'
  },
  // ... 其他搜索引擎
];

const SearchBox: React.FC = () => {
  const [searchQuery, setSearchQuery] = useState('');
  const [selectedEngine, setSelectedEngine] = useState('google');
  const [showDropdown, setShowDropdown] = useState(false);
  const [hoveredEngine, setHoveredEngine] = useState<string | null>(null);
  const containerRef = useRef<HTMLDivElement>(null);

  const currentEngine = searchEngines.find(e => e.id === selectedEngine) || searchEngines[0];

  // 处理搜索提交
  const handleSearchSubmit = () => {
    if (searchQuery.trim()) {
      const searchUrl = currentEngine.url + encodeURIComponent(searchQuery.trim());
      window.open(searchUrl, '_blank');
    }
  };

  // 处理 Enter 键
  const handleKeyDown = (e: React.KeyboardEvent) => {
    if (e.key === 'Enter') {
      handleSearchSubmit();
    }
  };

  // 选择搜索引擎
  const selectEngine = (engine: SearchEngine) => {
    setSelectedEngine(engine.id);
    setShowDropdown(false);
  };

  // 点击外部关闭下拉菜单
  useEffect(() => {
    const handleClickOutside = (event: MouseEvent) => {
      if (containerRef.current && !containerRef.current.contains(event.target as Node)) {
        setShowDropdown(false);
      }
    };

    document.addEventListener('mousedown', handleClickOutside);
    return () => document.removeEventListener('mousedown', handleClickOutside);
  }, []);

  return (
    <div className="search-section">
      <div className="search-section-container">
        <div className="search-container" ref={containerRef}>
          {/* 搜索引擎选择器 */}
          <div className="search-engine-selector-left">
            <div 
              className="current-engine" 
              onClick={() => setShowDropdown(!showDropdown)}
            >
              {currentEngine.icon}
            </div>
          </div>

          {/* 搜索框主体 */}
          <div className="search-main-area">
            <input
              type="text"
              value={searchQuery}
              onChange={(e) => setSearchQuery(e.target.value)}
              onKeyDown={handleKeyDown}
              placeholder={showDropdown ? '' : currentEngine.placeholder}
              className={`search-input ${showDropdown ? 'hide-text' : ''}`}
            />

            {/* 清除按钮 */}
            {searchQuery && !showDropdown && (
              <button
                onClick={() => setSearchQuery('')}
                className="clear-button"
                title="清除"
              >
                <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                  <line x1="18" y1="6" x2="6" y2="18"></line>
                  <line x1="6" y1="6" x2="18" y2="18"></line>
                </svg>
              </button>
            )}

            {/* 内部下拉菜单 */}
            {showDropdown && (
              <div className={`internal-dropdown dropdown-enter`}>
                <div className="engines-grid">
                  {searchEngines
                    .filter(e => e.id !== selectedEngine)
                    .map((engine, index) => (
                      <div
                        key={engine.id}
                        className="engine-option"
                        style={{ animationDelay: `${index * 0.05}s` }}
                        onClick={() => selectEngine(engine)}
                        onMouseEnter={() => setHoveredEngine(engine.name)}
                        onMouseLeave={() => setHoveredEngine(null)}
                      >
                        {engine.icon}
                      </div>
                    ))}
                </div>

                {/* 悬停提示 */}
                {hoveredEngine && (
                  <div className="engine-tooltip">{hoveredEngine}</div>
                )}
              </div>
            )}
          </div>

          {/* 搜索按钮 */}
          <button 
            onClick={handleSearchSubmit}
            className="search-button"
            type="button"
          >
            <i className="fa-solid fa-magnifying-glass"></i>
          </button>
        </div>
      </div>
    </div>
  );
};

export default SearchBox;
```

```javascript
    icon: 'fab fa-zhihu',
    url: 'https://www.zhihu.com/search?q=',
    placeholder: '在知乎中搜索...'
  }
];

let selectedEngine = 'google';
let showDropdown = false;

// 获取当前搜索引擎
function getCurrentEngine() {
  return searchEngines.find(e => e.id === selectedEngine) || searchEngines[0];
}

// 切换下拉菜单
function toggleDropdown() {
  showDropdown = !showDropdown;
  const dropdown = document.getElementById('internal-dropdown');
  const input = document.getElementById('search-input');
  
  if (showDropdown) {
    dropdown.classList.add('dropdown-enter');
    input.classList.add('hide-text');
    renderEngines();
  } else {
    dropdown.classList.remove('dropdown-enter');
    input.classList.remove('hide-text');
  }
}

// 渲染搜索引擎列表
function renderEngines() {
  const grid = document.getElementById('engines-grid');
  grid.innerHTML = '';
  
  searchEngines
    .filter(e => e.id !== selectedEngine)
    .forEach((engine, index) => {
      const option = document.createElement('div');
      option.className = 'engine-option';
      option.style.animationDelay = `${index * 0.05}s`;
      option.innerHTML = `<i class="${engine.icon} engine-icon"></i>`;
      
      option.onclick = () => selectEngine(engine);
      option.onmouseenter = () => showTooltip(engine.name);
      option.onmouseleave = () => hideTooltip();
      
      grid.appendChild(option);
    });
}

// 选择搜索引擎
function selectEngine(engine) {
  selectedEngine = engine.id;
  document.getElementById('current-engine-icon').className = engine.icon;
  document.getElementById('search-input').placeholder = engine.placeholder;
  toggleDropdown();
}

// 显示提示
function showTooltip(name) {
  const tooltip = document.getElementById('engine-tooltip');
  tooltip.textContent = name;
  tooltip.style.display = 'block';
}

// 隐藏提示
function hideTooltip() {
  const tooltip = document.getElementById('engine-tooltip');
  tooltip.style.display = 'none';
}

// 处理搜索提交
function handleSearchSubmit() {
  const input = document.getElementById('search-input');
  const query = input.value.trim();
  
  if (query) {
    const engine = getCurrentEngine();
    const searchUrl = engine.url + encodeURIComponent(query);
    window.open(searchUrl, '_blank');
  }
}

// 处理键盘事件
function handleKeyDown(event) {
  if (event.key === 'Enter') {
    handleSearchSubmit();
  }
}

// 清除搜索
function clearSearch() {
  const input = document.getElementById('search-input');
  input.value = '';
  updateClearButton();
}

// 更新清除按钮显示
function updateClearButton() {
  const input = document.getElementById('search-input');
  const clearBtn = document.getElementById('clear-button');
  
  if (input.value && !showDropdown) {
    clearBtn.style.display = 'flex';
  } else {
    clearBtn.style.display = 'none';
  }
}

// 监听输入变化
document.addEventListener('DOMContentLoaded', () => {
  const input = document.getElementById('search-input');
  input.addEventListener('input', updateClearButton);
  
  // 点击外部关闭下拉菜单
  document.addEventListener('click', (event) => {
    const container = document.querySelector('.search-container');
    if (!container.contains(event.target) && showDropdown) {
      toggleDropdown();
    }
  });
});
```

## 常见问题

### 1. 图标不显示？
**解决方案**：
- 检查 Font Awesome 是否正确引入
- 确认图标类名正确
- 检查网络连接

### 2. 下拉菜单无法点击？
**解决方案**：
- 检查 z-index 层级
- 确认 overflow 属性设置正确
- 检查点击事件绑定

### 3. 搜索无反应？
**解决方案**：
- 检查搜索 URL 是否正确
- 确认 `encodeURIComponent` 正确编码
- 检查浏览器控制台错误

### 4. 样式错乱？
**解决方案**：
- 检查 CSS 是否正确引入
- 确认没有样式冲突
- 使用浏览器开发者工具调试
